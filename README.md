# KNU Graduation Calculator

경북대학교 전자공학부 인공지능전공 학생을 위한 졸업학점 계산기입니다.

## 주요 기능

- 2023~2026학번 졸업요건 계산
- 전공, 교양, 전공필수, 남은 학점 현황 표시
- 부전공, 복수전공, 융합전공 학점 계산
- 강의 DB 기반 과목 검색 및 저장
- 외부 인정 학점 직접 추가

## 로컬 실행

정적 파일 기반 웹사이트라 별도 빌드 없이 실행할 수 있습니다.

```bash
python3 -m http.server 8088
```

브라우저에서 아래 주소를 열면 됩니다.

```text
http://127.0.0.1:8088/web/
```

## Vercel 배포

`vercel.json`에서 루트 접속을 `/web/`으로 리다이렉트하도록 설정되어 있습니다.

Vercel에서 GitHub 레포를 연결하면 별도 빌드 명령 없이 정적 사이트로 배포할 수 있습니다.
